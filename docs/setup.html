Enable pgvector extension
Windows: download pgvector from github.com/pgvector/pgvector → build or use prebuilt binaries.
Then in psql: CREATE EXTENSION vector;
Install
Create the Dexter database + user
psql -U postgres → CREATE USER dexter WITH PASSWORD 'dexter'; → CREATE DATABASE dexter OWNER dexter; → \c dexter → CREATE EXTENSION vector;
Config
